Description

A VipsStream is a source or sink of bytes for something like jpeg loading. It can be connected to a network socket, for example, or perhaps a node.js stream, or to an area of memory.

Subclass to add other input sources.

vips_streami_map ()

const void *
vips_streami_map (VipsStreami *streami,
                  size_t *length);

Map the stream object entirely into memory and return a pointer to the start. If length_out is non-NULL, the stream size is written to it.

This operation can take a long time. Use vips_streami_is_mappable() to check if a streami can be mapped efficiently.

The pointer is valid for as long as streami is alive.
